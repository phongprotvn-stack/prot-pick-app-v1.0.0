# Phase 3 — PWA + Mobile UX Tối Ưu

## Mục tiêu
Biến Prot Pick thành **PWA cài được lên màn hình chính**, load nhanh, dùng offline cơ bản.

## Tasks
1. **PWA Setup** — `vite-plugin-pwa` + manifest + service worker (cache-first cho assets)
2. **App Icons** — Tạo logo SVG/PNG đơn giản (PP shield icon)
3. **Offline Fallback** — Trang offline cơ bản khi không có mạng
4. **Mobile UX Tuning** — Pull-to-refresh, splash screen, viewport tối ưu
5. **Deploy & Verify** — Build, deploy Vercel, test PWA install prompt

## Không làm
- Push notifications (tốn phí, ko cần thiết)
- Complex sync (Firestore realtime đã cover)
